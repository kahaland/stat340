The goal of stat340 is to provide data and functions for the NMBU course Stat 340.

## Contents

Currently `stat340` contains the data sets

   * `bears`
   * `birth`
   * `bodydata`
   * `weights`
   * `yields` 
   * `yields.unstacked`
